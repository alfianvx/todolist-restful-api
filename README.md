# Todolist API

Todo API to store task in your day

- Endpoint: https://todolist-restful-api-production.up.railway.app/
- Documentation: https://documenter.getpostman.com/view/23075381/2s9YXk3fwb

## How to Run

1. Install Node JS
2. Run `npm install`
3. Run `npm run dev`
4. Open on http://localhost:3000

Library use in Repo

- Bcrypt
- Body Parser
- JWT
- Sequelize
- Nodemon
